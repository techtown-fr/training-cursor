# Historique, checkpoints & restauration

<div class="grid grid-cols-2 gap-8 mt-6">

<div>

### Garder la trace

<v-clicks>

- **Historique des chats & agents** : on retrouve une conversation passée
- **Checkpoints** : chaque étape de l'agent est un point de retour
- **Restauration** : revenir à un état du code en un clic
- **Reprise** : relancer un chat précédent pour continuer

</v-clicks>

</div>

<div>

### Pourquoi c'est précieux

<v-clicks>

- Expérimenter **sans peur** : on peut toujours revenir
- Comparer deux approches puis trancher
- Reprendre le fil d'une tâche **plusieurs jours après**
- Capitaliser sur le travail déjà fait

</v-clicks>

</div>

</div>

<div class="mt-6 p-3 bg-blue-50 rounded-xl text-sm text-center">

Réflexe : avant une grosse modif d'agent, repérez votre **point de restauration**.

</div>
